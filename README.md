# District Food
#### Мобильное приложение для просмотра ресторанов поблизости.
***
###### Целевая аудитория
Студенты МГТУ им. Н. Э. Баумана, которым нужна информация о местах
недалеко от университета, где они могли бы перекусить.
###### Функциональность приложения:
1. Поиск заведений поблизости, с последующим отображением в виде списка и возможностью перейти к детальному просмотру.
2. Просмотр страницы заведения
3. Оффлайн сохранение заведений и удаление сохраненных
4. Возможность авторизоваться, чтобы оставлять собственные отзывы
###### Экраны приложения
1. **“Главный”** <br/>
На главном экране осуществляется поиск заведений.<br/> 
По поисковому запросу выдаётся список ресторанов (в виде карточек), удовлетворяющих запросу. <br/> 
Сам поисковый запрос реализован в виде выпадающих списка окон. <br/>  <br/> 
<img src="https://github.com/K1selev/DistrictFood/blob/main/images/1.png" width="350" height="700">  &#xA0; &#xA0; &#xA0; <img src="https://github.com/K1selev/DistrictFood/blob/main/images/2.png" width="350" height="700"> <br/> <br/>
2. **“Заведение”**<br/>
На экране будет расположена информация о ресторане:
- фотографии
- название
- краткое описание
- адрес
- часы работы
- средний чек 
Есть возможность добавить данный ресторан в закладки, посмотреть чужие отзывы или добавить свой.
<br/><br/>
<img src="https://github.com/K1selev/DistrictFood/blob/main/images/3.png" width="350" height="700">  &#xA0; &#xA0; &#xA0; <img src="https://github.com/K1selev/DistrictFood/blob/main/images/31.png" width="350" height="700"><br/> <br/>
3. **“Закладки”** <br/>
На данном экране осуществляется переход к сохраненным заведениям. <br/>
Находятся они в виде списка названий и мини картинки рядом. <br/>
Также есть возможность удалить каждую отдельную закладку при нажатии на иконку сердце от названия. <br/> <br/>
<img src="https://github.com/K1selev/DistrictFood/blob/main/images/4.png" width="350" height="700"> <br/> <br/>
4. **“Профиль”** <br/>
На данном экране есть возможность изменить свои фото и имя. <br/>
Также можно посмотреть оставленные ранее комментарии <br/>
Если пользователь пользуется приложением первый раз или просто не зашел в свой аккаунт, то открывается регистрация. В ином случае сразу откроется профиль
<br/> <br/>
<img src="https://github.com/K1selev/DistrictFood/blob/main/images/6.png" width="350" height="700">  &#xA0; &#xA0; &#xA0; <img src="https://github.com/K1selev/DistrictFood/blob/main/images/61.png" width="350" height="700"> <br/> <br/>
5. **“Карты”** <br/>
Показывает ближайшие рестораны поблизости <br/>
Можно нажать на них и проложить маршрут для нужного ресторана <br/> <br/>
